# datasetAlgorithms
Repository of newly created datasets for Algorithms paper

The repository consists of 4 newly created datasets:

 1. Synthetic Gaussian Data sigma=1.0
 2. Synthetic Gaussian Data sigma=2.0
 3. Synthetic 3D Moon Data
 4. 10D, Five-Class Spiral Benchmark Dataset

The other datasets are publicly available: 
Breast Cancer Wisconsin (Diagnostic) Dataset, Wine Dataset, Reduced MNIST Dataset are part of scikit-learn Python installation.
  
Forest Cover Type (7Classes) Dataset is available at https://archive.ics.uci.edu/dataset/31/covertype at the time of this writing.
